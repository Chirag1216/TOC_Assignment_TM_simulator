About the Concept

A Turing Machine is a 7-tuple M = (Q, Σ, Γ, δ, q₀, qₐ, qᵣ) — a mathematical model that defines what is computable. It has an infinite tape, a read/write head, and a finite set of states.

At each step, based on the current state and symbol read, the transition function δ determines the new state, the symbol to write, and whether the head moves Left, Right, or Stays.

Try presets above, or write your own rules. The simulator halts when it enters an accept/reject state or when no rule matches.
